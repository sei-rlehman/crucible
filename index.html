    Crucible: A Cyber Simulation Framework

    Crucible delivers cost-effective, scalable, and dynamic virtual environments that feature a modeled topology, simulated user activity, and scenario events to improve performance of mission-essential tasks or individual qualification requirements.


    ● Open-source cyber simulation framework
    ● Modular design featuring extensive APIs
    ● Customizable, immersive, browser-based user interface
    ● “Infrastructure as code” approach to topology building
    ● Flexible integration of powerful, third-party open-source tools
    ● Scenario-based simulation and assessment
    ● Support for multiple on-prem and cloud virtualization platforms
    ● Efficiency through automation
    ● Interoperability through open standards

    The Framework


    The Crucible Framework applications work together to provide simulation developers with the tools they need to develop high quality cyber security training events

    Each application is designed around a robust API that encourages interoperability with other Crucible Framework applications as well as powerful third-party tools.


    Crucible Toolset


    ● Crucible facilitates training development by easily integrating with industry leading software as well as well known open source solutions

    ● Run your training on multiple infrastructure providers, on-prem or in the cloud

    ● Crucible supports Keycloak for identity and access managment
